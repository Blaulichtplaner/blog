Pro Standort können Sie mehrere Arbeitsbereiche festlegen. Mit dem Erstellen von einem Standort erstellen Sie automatisch auch den ersten Arbeitsbereich. Legen Sie bei Bedarf weitere Arbeitsbereiche über den Button "neuer Arbeitsbereich" an. Hierzu müssen Sie diesen nur benennen, eine individuelle Farbe zuteilen und eine Basisvergütung für diesen Arbeitsbereich festlegen. Nutzen Sie unterschiedliche Farben, um später einen übersichtlichen Dienstplan zu erhalten. Denken Sie gegebenenfalls auch gleich wieder an die Erstellung von Arbeitszeit-Vorlagen.

### Vergütung

In dem Bereich "Vergütung" können Sie die Vergütung festlegen, die Sie mit den Mitarbeitern für diesen
Arbeitsbereich festgelegt haben. Alternativ können Sie auch pro Mitarbeiter eine individuelle Vergütung festlegen, die dann maßgäblich für die Berechnung ist. Wird keine individuelle Vergütung festgelegt, dann gilt die Vergütung, die hier im Arbeitsbereich definiert ist.

Sie müssen mindestens die Vergütung pro Stunde definieren. Zusätzlich können Sie die Einsätze vergütungen und 
eine Sondervergütung an Wochentagen (z.B. Sonntag) und an Feiertagen definieren. 

Wenn sich ab einen bestimmten Zeitpunkt die Vergütungssätze ändern, dann erstellen Sie einen neuen Abrechnungsdatensatz und legen das Datum fest, ab wann die neuen Vergütungssätze gelten.

### Kostenträger

Um die geleisteten Dienste mit einem Kostenträger abzurechnen können Sie einen Arbeitsbereich einem Kostenträger-Vertrag zuordnen. Diese Verträge müssen Sie zuerst in der Firmen-Verwaltung im Bereich "Kostenträger" erstellen.
