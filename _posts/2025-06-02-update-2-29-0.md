In der Version 2.29.0 wurde die Möglichkeit geschaffen, Pausen zu erfassen und eine monatliche Statistik für die Dienstpläne abzurufen.

### Erfassung von Pausen 

Der Manager kann jetzt pro Arbeitsbereich festlegen, ob Pausen erfasst werden sollen. Dabei kann eine Standardpause festgelegt werden, die dann in der Auswertung automatisch hinzugefügt wird. 
Die Mitarbeiter können dann bei der Auswertung weitere Pausen hinzufügen. Die Pausen werden in der aktuellen Version noch nicht für die Abrechnung berücksichtigt, so dass sie als bezahlte Pausen gelten.

![screenshot80](/assets/screenshots/pausenerfassung.png)

### Monatliche Statistik für die Dienstpläne 

In der Dienstplanung finden Sie jetzt einen Button zur monatlichen Statistik. Darin sehen Sie die Anzahl der geplanten und zugewiesenen Stunden. Sie können die Statistik für einen Monat oder für mehrere Monate eines Jahres abrufen. Bitte beachten Sie, dass die Statistik hauptsächlich für die laufende Planung gedacht ist. D.h. es werden maximal die letzten 2 Monate angezeigt.

![screenshot80](/assets/screenshots/dienstplan_statistik.png)

### Weitere Änderungen und Korrekturen

- In der Liste der unbesetzten Dienste werden jetzt auch Dienste angezeigt, die bereits begonnen haben
- Die Funktion zum Diensttausch kann pro Standort deaktiviert werden
- Timeout Probleme bei der Zuweisung von Mitarbeitern behoben
